# 🛰️ Satellite Image Land-Use Classifier & Temporal Change Detector

A complete computer vision system that classifies land-use types from satellite imagery and detects land-cover changes between two time periods.

## 🧪 Model Architecture

```
ResNet-18 (ImageNet pretrained)
  Phase 1: Backbone FROZEN  → train FC head only (3 epochs, LR = 1e-3)
  Phase 2: layer3 + layer4 + FC UNFROZEN (5 epochs, LR = 1e-4)

Classifier head: Dropout(0.3) → Linear(512, 10)

Change Detector:
  Strip FC → 512-dim embedding per tile
  Cosine similarity(T1_emb, T2_emb)
  Threshold via Youden's J on ROC curve
  Flag as CHANGED if similarity < threshold
```

## 🔧 Troubleshooting

| Problem | Fix |
|---------|-----|
| `ModuleNotFoundError` | Activate venv and run `pip install -r requirements.txt` |
| `resnet18_best.pt not found` | Run notebook 03 first |
| Slow training | Reduce `BATCH` size; set `num_workers=0` on Windows |
| EuroSAT download fails | Download manually from https://madm.dfki.de/files/sentinel/EuroSAT.zip → extract to `data/EuroSAT_RGB/` |
| UC Merced not found | Download from http://weegee.vision.ucmerced.edu/datasets/landuse.html → place in `data/UCMerced_LandUse/Images/` |
| UMAP ImportError | `pip install umap-learn` |
